API

- API: node.JS with typescript
- FRONTEND: react with typescript and Mongodb

1. cd api-node
2. npm install
3. edit .env file
    - Add port
    - Add mongodb url (ex: mongodb://localhost:27017/ux-studio-challenge)
4. npm start (or npm run start:nodemon for nodemon)

FRONTEND

1. cd frontend
2. npm install
3. edit .env file
    - add api URL
